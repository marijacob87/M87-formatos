# M87 • Calculadora de Formatos

Ferramenta para cálculo de aproveitamento de papel e montagem de formatos para impressão.

## Funcionalidades

- Melhor montagem
- Segunda melhor montagem
- Rotação automática das peças
- Margem de segurança
- Espaçamento entre peças
- Visualização gráfica da montagem

Desenvolvido com Python + Streamlit.
